---
title: openeye serve
---

Start a FastAPI inference server.

```bash
openeye serve yolov8
openeye serve yolov8 --port 9000
```

### Options

| Flag | Description |
|------|-------------|
| --host <addr> | Bind address (default: 0.0.0.0) |
| --port <num> | Port number (default: 8000) |
| --variant <name> | Serve a specific variant |
