---
title: openeye pull
---

Download model weights from HuggingFace Hub.

```bash
openeye pull yolov8
openeye pull depth-anything
openeye pull grounding-dino
```

### Options

| Flag | Description |
|------|-------------|
| --variant <name> | Pull a specific variant (e.g., quantized) |
| --force | Re-download even if already present |
